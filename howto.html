How To Use

Encryption:
1. Click on "Encrypt" at the title of the home page.
2. Enter a key of any size. Make a mental note of this!
3. Select the .txt file you want to encrypt.
4. Hit the submit button. The encrypted file will be downloaded automatically. Now you can store the encrypted file instead of the original file! Decrypt this file whenever you wish to access it. But do remember your key!


Decryption:
1. Click on "Decrypt" at the title of the home page.
2. Enter the key you used to encrypt the file.
3. Select the encrypted file you want to decrypt.
4. Hit the submit button. The decrypted file will be downloaded automatically. You can now read this file!
